# Дипломный проект

## База знаний (онлайн школа)

### MVP

- клиентский интерфейс
  - выбрать дисциплину
  - посмотреть разделы
  - посмотреть текстовые, видео материалы
  - скачивать задания и доп материалы
  - возможность зарегистрироваться
  - добавлять материалы в избранное
  - отмечать выполненные задания, прикреплять файлы
  - смотреть рейтинги на сайте
- админский интерфейс
  - модерация контента на сайте
  - управление пользователями сайта
  - создание пользователей с разными ролями (админ, модератор, пользователь)
